# CircuitToolsMT — Política de Privacidad

Política de privacidad de la app **CircuitToolsMT** (editor de circuitos esquemático + PCB para Android y Linux).

Los circuitos se editan y guardan localmente. La app usa **Firebase Analytics** para estadísticas de uso anónimas (sin datos personales ni contenido de proyectos).
